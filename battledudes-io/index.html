About the game

About BattleDudes.io

BattleDudes.io is a real-time multiplayer 2D shooter built around destructible maps, team fights, and long-term progression. Instead of a single scripted run, the game pushes players into repeated battles where map destruction and loadout growth matter.

Why it fits

Why this mission belongs in the library

It belongs in the library because it captures the browser-shooter audience from a different angle. The top-down 2D view changes the feel, but the weapon unlocking, team combat, and constant conflict still serve the same player intent.

Highlights

What players can expect

  • Fully destructible maps keep matches dynamic.
  • XP and unlockable weapons reward long sessions.
  • Leaderboards strengthen the competitive hook.

Game controls

Controls

  • W A S D: move
  • R: reload
  • E: enter car
  • Left click: shoot
  • Right click: emote
  • Mouse wheel or 1-4: switch weapons
