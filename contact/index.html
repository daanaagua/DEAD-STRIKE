When to write

Use the contact path for issues that affect page quality or site trust.

  • Embedded game no longer loads or opens the wrong route.
  • Page title, summary, schema, or category placement needs correction.
  • Policy, privacy, or removal questions tied to a specific page.
  • Metadata or crawl signals that look broken after a deploy.

What to include

A little context makes issue review much faster.

  • The exact page URL.
  • The browser and device if the issue is technical.
  • A short note describing what you expected to happen.
  • A screenshot or repro steps when available.

Current public route

The site currently handles public requests through the project repository.

If a more direct mailbox or dedicated support workflow is published later, this page will be updated here first. Until then, use the repository link and mention the affected DEAD STRIKE page clearly.
