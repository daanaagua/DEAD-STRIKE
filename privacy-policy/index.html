Privacy Policy

This site is static, but it still uses analytics and third-party game hosts.

DEAD STRIKE is a browser-based static site. Most pages contain text, images, structured data, and internal links. Some pages also embed third-party game hosts, and the site uses Google Analytics to measure basic traffic and page usage. This page explains those practical data paths in plain language.

Main data paths

Analytics and embeds

The key privacy considerations on this site come from analytics scripts and embedded game frames, not from local user accounts.

Google Analytics Third-party embeds No local account system

What the site itself uses

The site currently loads Google Analytics on its pages.

DEAD STRIKE includes a Google Analytics tag from www.googletagmanager.com. That means page visits, basic technical details, and interaction signals may be processed by Google Analytics under Google's own policies.

What embedded pages use

Playable pages may also send requests to external game hosts.

Some playable routes load content from hosts such as dead-strike.1games.io and html5.gamemonetize.co. When an embedded frame or direct game tab loads, those hosts may receive browser and device information needed to serve the game.

What the site does not do

There is no local account, checkout flow, or user profile system on the site.

  • There is no user registration flow on DEAD STRIKE.
  • The site does not ask visitors to create a local password.
  • No first-party user profile dashboard is published on the current site.

Your options

If you do not want third-party game hosts to load, stay on non-play pages only.

  • Read category, policy, and informational pages without opening embeds.
  • Avoid opening playable iframes or direct game tabs if you do not want those hosts contacted.
  • Use the contact page for page-specific privacy questions.
